minimaal: why another static site generator?

MAY 14, 2018

Why minimaal?

Simplicity

minimaal is simple

Elegance

minimaal is beautiful

Speed

minimaal is fast

Features

  • Posts written in markdown, no need for messy HTML or clunky CMS
  • No need to worry about databases
  • Focus on writing content, not on configuring extensions
  • Syntax highlighting thanks to pygments

Credits

  • markdown2 for markdown rendering
  • pygments for syntax highlighting
  • Jinja2 for templating
  • milligram for stylesheets
